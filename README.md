# jazz-wednesdays-data

Τα ζωντανά δεδομένα της εφαρμογής **Jazz Τετάρτες στο Χαλάνδρι**.
Η εφαρμογή διαβάζει το `points.json` από εδώ σε κάθε άνοιγμα — **κάθε αλλαγή σε αυτό το αρχείο εμφανίζεται σε όλα τα κινητά αυθημερόν, χωρίς νέο build**.

## Πώς κάνω αλλαγή

1. Άνοιξε το `points.json` (μπορείς και από το κινητό, μέσω github.com → Edit ✏️).
2. Κάνε την αλλαγή και **Commit**.
3. Μέσα σε λίγα λεπτά, κάθε άνοιγμα της εφαρμογής παίρνει τα νέα δεδομένα.

## Τι μπορώ να αλλάξω

**Νέο μαγαζί στο πρόγραμμα** — πρόσθεσε ένα αντικείμενο στη λίστα:

```json
{
  "id": "live_XX",
  "categoryId": "live_sessions",
  "name": "Όνομα Μαγαζιού",
  "description": "Δυο λόγια για τον χώρο.",
  "address": "Οδός 12, Χαλάνδρι",
  "hours": "Τετ–Σαβ 20:00–01:00",
  "phone": "210 1234567",
  "latitude": 38.0225,
  "longitude": 23.7995,
  "isLive": false
}
```

Κατηγορίες: `live_sessions` (Ζωντανές Σκηνές) · `spins` (DJ Επιλογές) · `mood` (Jazz Ακροάσεις) — και όλες οι δημοτικές (`mun_*`, `parking`).

**«Ζωντανά τώρα»** — βάλε `"isLive": true` στο μαγαζί που παίζει απόψε (και ξανά `false` μετά). Στον χάρτη εμφανίζονται οι κύκλοι live.

**Πρόγραμμα εβδομάδας** — πρόσθεσε `program` σε ένα μαγαζί· εμφανίζεται στην οθόνη «Πρόγραμμα»:

```json
"program": [
  { "day": "Τετάρτη", "time": "21:00", "title": "Live jazz τρίο" },
  { "day": "Παρασκευή", "time": "22:00", "title": "DJ set — soul & funk" }
]
```

Ημέρες ακριβώς έτσι: `Δευτέρα` `Τρίτη` `Τετάρτη` `Πέμπτη` `Παρασκευή` `Σάββατο` `Κυριακή`.

**Χορηγός μαγαζιού** — πρόσθεσε `"sponsor": "Όνομα Χορηγού"` και το δελτίο του μαγαζιού δείχνει «Με την υποστήριξη: …».

## Προσοχή

- Το αρχείο πρέπει να μένει **έγκυρο JSON** (κόμματα, εισαγωγικά). Πριν το commit, ένα πέρασμα από https://jsonlint.com βοηθά.
- Αν το αρχείο χαλάσει, η εφαρμογή συνεχίζει με τα τελευταία ενσωματωμένα δεδομένα — τίποτα δεν κρασάρει, απλώς οι αλλαγές περιμένουν τη διόρθωση.
